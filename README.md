# 校园助手

项目结构基于饿了么大大的[iReading](https://github.com/attentiveness/reading.git)，本是用[ignite](https://github.com/infinitered/ignite)构建基本结构，但觉得还是iReading的结构比较清晰。

### 功能
* 论坛
* 比赛组团
* 社团
* 二手市场
* 校园公告

### 技术栈
* react-native
* redux
* react-navigation
* redux-saga
* codePush
